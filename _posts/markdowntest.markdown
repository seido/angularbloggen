## コード
4つ以上インデントする

[指定例]

```
    function a() {
        var i = 0;
    }
```

[表示例]

    function a() {
        var i = 0;
    }

もしくは```を使用する

[指定例]

    ```
    hoge
    ```

[表示例]

```
hoge
```

## リスト

[指定例]

`*`を使う

```
* hoge
* fuga
```

`+`を使う

```
+ hoge
+ fuga
```

`-`を使う

```
- hoge
- fuga
```

[表示例]

- hoge
- fuga

## 数字付きリスト

[指定例]

```markdown
1. hoge
2. fuga
```

[表示例]

1. hoge
2. fuga
